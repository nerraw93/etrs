# Client Batch - search

#### (**Client Only**)

Search for batches belonging to a client by its id.

**URL** : `/api/v1/clients/{clientId}/batches/search`

**Query Params**

1. `clientId` = **Required**. ID of the client to whom the batches belong.
2. `id` = **Required** ID of the batch.
3. `status` = **Required** Status of the batch. `draft`, `sent`, `processing`, or `finish`

**Method** : `GET`

**Auth required** : YES

## Success Response

**Code** : `200 OK`
