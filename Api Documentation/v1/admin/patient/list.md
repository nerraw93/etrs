# Patient list

#### (**Admin Only**)

Used to get list of patients

**URL** : `/api/v1/patients`

**Query Params**

1. `count` = Number of batch to be returned. Defaults to `30`
2. `page` = Page number. Defaults to 1.

**Method** : `GET`

**Auth required** : YES

## Success Response

**Code** : `200 OK`
