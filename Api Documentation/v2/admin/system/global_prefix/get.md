# Get admin global prefix

#### (**Admin Only**)

Get admin global prefix

**URL** : `/api/admin/system/global-prefix`

**Method** : `GET`

**Auth required** : YES

## Success Response

**Code** : `200 OK`

**Content examples**

```json
{
    "success":true,
    "message":"",
    "global_prefix": "OL",
}
```
